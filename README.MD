
# OurAir

Sebuah aplikasi web yang menyediakan layanan penerbangan pesawat


## fitur

- authentifikasi
- pencarian penerbangan
- detail penerbangan
- detail airport
- pemesanan penerbangan
- notifikasi
- pembayaran menggunakan midtrans
- seeder
